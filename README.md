# AWS EKS Terraform Deployment

Terraform module for deploying a containerized service to AWS EKS with complete infrastructure including VPC, networking, and security configurations.

## Overview

This repository contains a Terraform module that provisions:
- AWS EKS Kubernetes cluster
- VPC with public and private subnets
- Networking components (Internet Gateway, NAT Gateway, route tables)
- Security configurations (security groups, IAM roles)
- Containerized service deployment

## Prerequisites

- AWS CLI configured with appropriate credentials
- Terraform >= 1.0
- kubectl installed
- AWS IAM permissions for EKS, VPC, EC2, and related services
